CSS FÁCIL DE LEER

En un archivo css es muy important que tomemos en cuentas estos mecanismos para poder un css que sea fácil de leer.

Espacios en blanco
Debemos tener en cuenta los espacios en blanco, tabulaciones, colocaciones de llaves y colocaciones de par clave valor.
Comentarios
También añadir comentarios a nuestros archivos css, ayuda mucho para documentar o explicar partes de nuestro código.
Agrupacion de selectores
